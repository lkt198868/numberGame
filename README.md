# 数字求和游戏

这是一个简单的数字求和教育游戏，适合儿童学习基础加法。

## 游戏规则

1. 选择一些数字卡片，使它们的和等于目标数字
2. 点击"检查答案"按钮验证你的答案
3. 答对后会有庆祝动画

## 游戏特点

- 目标数字最大为10
- 随机数字范围为1-9
- 支持桌面和移动设备（包括微信）
- 有趣的动画和交互效果

## 技术实现

- 纯HTML、CSS和JavaScript实现
- 响应式设计，适配各种屏幕尺寸
- 支持触摸事件，适合移动设备使用
